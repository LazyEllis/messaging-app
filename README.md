# Messaging App

This is a web app for a messaging app. It allows users to send messages to each other either as DMs to a single user or in group chats with multiple users.

## Features

- User Authentication
- Profile Customization
- Sending DMs to users
- Creating of and sending messages in Group Channels

## Installation

1.  Clone the repository

    ```bash
    git clone git@github.com:LazyEllis/messaging-app.git
    ```

2.  Install the project's dependencies

    ```bash
    npm install
    ```

3.  Create a .env file and define environment variables

    ```env
    VITE_API_URL="http://localhost:3000"
    ```

4.  Run the development server

    ```bash
    npm run dev
    ```
